**AddressMe is a desktop application for managing your destination details.** While it has a GUI, most of the user interactions happen using a CLI (Command Line Interface).

* If you are interested in using AddressMe, head over to the [_Quick Start_ section of the **User Guide**](UserGuide.html#quick-start).
* If you are interested about developing AddressMe, the [**Developer Guide**](DeveloperGuide.html) is a good place to start.
